# Transform Editor

This widget can be used to test a `transform` action.

## Model

```json
{
    "type": "transformeditor",
    "completeMsgObject": false,
    "input": {},
    "transform": {
        "type": "transform",
        "aggregateOne": [
            {}
        ]
    }
}
```

See [actions](../../actions/README.md) for more details about the `transform` action.
